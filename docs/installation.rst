Installation
============
``cv_pro`` can be installed directly from the repo using pip::

    pip install git+https://github.com/dd-hebert/cv_pro.git


Uninstall
---------
To uninstall ``cv_pro`` and remove all files associated with it, run the following commands::

    pip uninstall cv_pro

The ``cv_pro`` configuration files are located ``.config/cv_pro`` which can be found in your home directory.
You can safely delete the ``cv_pro`` folder inside ``.config`` at any time.
